# 🛡️ FakeGuard AI - Fake News Detector

Advanced Transformer-based Fake News Detection App with beautiful UI and Lottie animations.

## How to Run
1. pip install -r requirements.txt
2. Download WELFake dataset from the below link and put in `data/` folder

----->  https://drive.google.com/file/d/1fW8CPQ-B4VTfz-9P-X_2gETDRq8vIaXX/view?usp=drive_link
2. Download model folder the given google drive link and put in `model/` folder

----->  LINK :   https://drive.google.com/drive/folders/1_amjmf37gUgGPssS2kIn6ScYgOAcLt6Y?usp=drive_link


3. python train_model.py
4. streamlit run app.py
